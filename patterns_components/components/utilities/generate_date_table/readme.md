# Generate Date Table

Generates a range of dates (from `start_date` to `end_date`) and useful
attributes on them:

 - date
 - year
 - month
 - day
 - day_of_week
 - day_of_week_name
 - week
 - quarter
 - year_half
 - is_business_day
 - is_month_start
 - is_month_end
 - is_quarter_start
 - is_quarter_end
 - is_year_start
 - is_year_end
 - is_leap_year
